# ERC-20
setUp Project 
```bash
requirement: solidity >=0.4.22 <0.9.0, nodejs >= 10.x, truffle 5.3.0
```

Create new project
```bash
truffle init
```

Run Ganache network
```bash
ganache-cli
```

Compile new code
```bash
truffle compile
```

Migrate project with network
```bash
truffle migrate --network [network_name]
```

Login to truffle console
```bash
truffle console --network [network_name]
```
